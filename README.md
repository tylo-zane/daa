# Design Against Anxiety
## Zane Marcinyshyn
*Work-in-progress version will [go live here](https://tylo-zane.github.io/daa/).*

Over 19.1 percent of adults in the U.S. are diagnosed with an anxiety disorder[(Leber, 2020)](https://level-level.com/blog/inclusive-design-designing-for-people-with-anxiety/). Design Against Anxiety is a tool for user interface designers to help mitigate the anxiety experienced by their users.
